# acidbase
Simulation of an acid/base titration using Bokeh and numpy
## Prerequisites
'''
python3
Bokeh
Numpy
'''

### Installing

Download the code from github

### Running
Change directory to the folder containing acid_base.py. In the command line type:
'''
bokeh serve --show acid_base.py
'''
